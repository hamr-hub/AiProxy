# 安全指南

## 敏感信息清单

以下信息必须加密存储，禁止明文记录：

- API Key / Token
- 密码
- 证书 / 私钥
- OAuth 凭证

## 禁止事项

- ❌ 日志/埋点打印敏感字段
- ❌ 把敏感字段写入可被前端读取的 storage
- ❌ 在代码中硬编码密钥

## 脱敏规则

```javascript
// ✅ 正确：只显示前 4 位
logger.info(`API Key: ${apiKey.substring(0, 4)}***`);

// ❌ 错误：打印完整密钥
logger.info(`API Key: ${apiKey}`);
```

## 建议项

- 对外跳转 URL 白名单
- 错误上报脱敏
- 使用环境变量管理密钥
